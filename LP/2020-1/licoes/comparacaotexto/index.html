O Python faz comparações de textos (string) com base na ordem lexicográfica - isto é, a ordem do dicionário. Isso é utilizado, por exemplo, para colocar os seus contatos em ordem na agenda do celular.

Os símbolos < e >, quando comparamos strings, querem dizer "a string vem antes ou depois no dicionário?". Se vem antes é menor, se vem depois é maior.

print('a' < 'f')
resultado: True, o que quer dizer que 'a' vem antes de 'f'

print('a' > 'f')
resultado: False, o que quer dizer que 'a' não vem depois de 'f'

print('s' > 'm')
resultado: True, o que quer dizer que 's' vem depois de 'm'

print('x' > 'z')
resultado: False, o que quer dizer que 'x' não vem depois de 'z'

Para saber se uma string é "maior" ou "menor" que outra o Python vai comparar as duas strings caractere por caractere.

Ou seja, ele compara o primeiro caractere de uma string com o primeiro caractere da outra string. Se um caractere for "menor" que o outro, então aquela string é "menor" também. Se forem iguais, o Python passa para o próximo caractere: compara o segundo caractere de uma string com o segundo caractere da outra string
