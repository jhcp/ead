O resultado de uma função nada mais é do que um valor. Neste caso, é um valor numérico.

Assim como qualquer outro valor, ele não irá aparecer na tela a não ser que você use a função print

print(minhaFuncao(10))

Opa, quer dizer que print também é uma função?!?

Sim! O Python já tem váaaaarias funções pré-definidas. A definição delas existe, mas elas já vêm junto quando instalamos o Python. Por isso podemos usar print(), len(), input(), int(), float(), etc. sem muito stress.

Ainda bem, né?

Uma função é um bloco de comandos que pode ser reutilizado depois em um programa. Elas são identificadas por nomes, que seguem as mesmas regras dos nomes de variáveis

Uma das coisas legais de funções é que eu posso definir a função uma vez só e depois usa-la dezenas, centenas, milhares, milhõoooes de vezes! Basta usar o nome dela e dizer o valor do(s) parâmetro(s)

#definição da função:
def f(x):
  return x**2-1

#chamada da função:
print(f(2))#dá 3
print(f(3))#dá 8
print(f(0))#dá -1
print(f(-1))#dá 0
print(f(10))#dá 99
anatomia de uma função: 
def nome(parametros):
  corpo da função
  return resultado

exemplo: 
def raizQuadrada(x):
  return x**(1/2)
  nome: raizQuadrada
  parâmetro: x
  resultado: x**(1/2)
