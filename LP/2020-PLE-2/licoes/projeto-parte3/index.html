Equipe

O projeto pode ser feito individualmente, em dupla, ou em trio.

Entrega

Apenas um(a) representante da equipe irá entregar o arquivo do projeto, pelo SIGAA. O nome do arquivo deve ser o nome completo da pessoa que entraga-lo.

Pode mudar a equipe?

Sim, pode mudar a equipe - se alguém tiver se encostado, se alguém abandonou a disciplina, se não é possível mais conciliar a agenda de vocês, ou por qualquer outro motivo.

O detalhe é que, mudando a equipe, você deve mudar os dados também. Por exemplo, se a sua parte 1 é sobre jogos, você pode manter o mesmo código mas substituir os jogos para não correr o risco de ficar igual.

Como salvar o arquivo

Quando terminar de fazer o código no repl.it, abra o bloco de notas, copie o código para ele, e salve com o NOME_COMPLETO_DA_PESSOA_QUE_VAI_ENTREGAR.py, escolhendo o Tipo "Todos os arquivos", conforme imagem abaixo. É esse arquivo que será entregue pelo SIGAA.

Como tirar dúvidas?

Faça as perguntas pelo grupo do WhatsApp mas não envie o seu código (não envie), para evitar o risco de alguém copia-lo e acabar lhe prejudicando.
