Bissexto

  • São bissextos todos os anos múltiplos de 400, p.ex: 1600, 2000, 2400, 2800...
  • São bissextos todos os múltiplos de 4, exceto se for múltiplo de 100 mas não de 400, p.ex: 1996, 2000, 2004, 2008, 2012, 2016... O ano 1900, por exemplo, não é bissexto, pois ele é múltiplo de 100 mas não de 400.
  • Não são bissextos todos os demais anos.

Bombons

Não é necessário usar o if neste problema.

Fique atento que o resultado precisa incluir o texto exatamente como está escrito no enunciado, com V maiúsculo. Exemplo: Valor: R$17.00

Se o seu programa imprimir só 17.00, ou valor: R$17.00, ou Valor: R$ 17.00, ou Valor: R$17, por exemplo, a resposta não será aceita.

Somatório de lista

Este problema já tem a parte inicial programada, que cria uma lista com os valores digitados pelos usuários. A lista está armazenada na variável numeros.

O seu trabalho, aqui é completar o código calculando a média dos números da lista, que deve ser impressa na tela com 2 casas decimais.

Para calcular a média, você pode usar as funções ensinadas na Semana 5.

Lembre-se de escolher a opção Python 3 quando for escrever a resposta no The Huxley.

Quando só alguns dos seus casos de teste estão errados, quer dizer que seu programa não está dando sempre o resultado correto. Por exemplo, no problema de formas geométricas, alguns alunos erraram a conta do perímetro do retângulo - para todas as outras formas, o resultado estava correto.

Texto adicional

Se lembre que o que irá aparecer na tela tem que ser exatamente igual à saída esperada, sem nada a mais nem a menos. Por exemplo, normalmente nós faríamos um programa que calcule o dobro de um número assim:

x = float(input('digite um número')) print('O dobro de', x, 'é', x*2)
Mas para a ferramenta de correção automática, precisaremos escrever assim, sem os textos:
x = float(input()) print(x*2)
a não ser que o enunciado do problema peça para incluir algum texto.

2 casas decimais

Alguns problemas podem pedir que você mostre o resultado com exatamente duas casas decimais. Você certamente já aprendeu a fazer isso com os vídeos do Guanabara, mas aqui vai um exemplo para facilitar a sua vida:

resultado = 3**0.5 print('{:.2f}'.format(resultado))
